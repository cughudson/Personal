中建科技馆BIM实施

基本信息

  1. 项目名称：中建光谷之星（E 地块）中国建筑科技馆幕墙工程
  2. 建筑面积：约 1.5 万平方米
  3. 建筑高度：58.5 米
  4. 建设单位：中建三局投资集团有限公司
  5. 施工单位：湖北中南钢结构有限公司
  6. 结构形式：钢结
  7. 建设地址：湖北省·武汉市·东湖新技术开发区·高新大道与光谷 6 路交汇处
  8. 实施时间:2019年3月 ~ 2019年5月

建成后的中国建筑科技馆（湖北·武汉）

重难点介绍

中建光谷之星（E 地块）中建科技馆项目（下面称“中建科技馆项目”）主要由裙房及位 于场地中间的“种子”，其中裙房为带“侧弯”的直立锁边屋面及石材山墙构成。“种子”部 分则由双曲异形不锈钢板面和位于种子中心部分的球面双曲玻璃构成。根据现场施工部署及工 作安排，不锈钢种子部分为整个项目的进度控制要点。同时，不锈钢种子部分除南北两侧包含 部分不规则双曲异形玻璃外，其余全部有异形不锈钢板构成。生产加工难度大，是整个项目的 下料及施工难点，基于此，本总结主要介绍 BIM 在种子部分双曲异形不锈钢板的施工下料方法。

为保证施工效果，业主花费数百万制作的1：6"模型"

批量*stp文件导出

在项目实施过程中，分包厂家要求将每块双曲异形面板以LOD400的精度导出成符合工业生产需求的*.stp格式的三维模型， 而整个项目共有三千来块完全不同的不锈钢异形板块，首先想到的是增加BIM实施人员，让BIM实施人员手动依次导出板块，但此种方法不但难以保证导出的准确率 假如后期模型出现变更，工作人员将不得不重新依次导出板块，难以满足项目的施工工期不说项目BIM实施成本将随着模型变更次数的增多而呈现指数级别的增加，这完全背离了幕墙项目使用BIM的初衷。在这种背景下 ，项目团队通过深入分析、梳理此项目的BIM实施流程，使用C#并结合Grassshopper研发出了能将模型自动导出成*stp格式文件的程序模块，并将其整合在了Grashopper中。使用程序替代人工操作，给模型的导出工作提供了极大的便利。间接为项目BIM实施节约了成本。

  private void RunScript(string ParentLayerName, string Dir, bool Toggle, ref object A)
  {
    //Get target layer
    if(!Toggle)return;
    Layer ParentLayer = Rhino.RhinoDoc.ActiveDoc.Layers.FindName(ParentLayerName);
    if(ParentLayer == null)return;
    Layer[] a_ChildrenLayer = ParentLayer.GetChildren();
    if(a_ChildrenLayer == null){
      A = ParentLayer.Name + " :不包含子图层";
      return;
    }
    //Deselect All Object
    Rhino.DocObjects.RhinoObject[] Objs_Temp = Rhino.RhinoDoc.ActiveDoc.Objects.GetSelectedObjects(false, false).ToArray();
    foreach(var obj in Objs_Temp){
      obj.Select(false);
    }
    //select target Object
    for(int Index = 0; Index < a_ChildrenLayer.Length;Index++){
      Rhino.DocObjects.RhinoObject[] Objs = Rhino.RhinoDoc.ActiveDoc.Objects.FindByLayer(a_ChildrenLayer[Index]);
      for(int j = 0; j < Objs.Length;j++){
        Objs[j].Select(true);
      }
    }
    //export selected object to stp object
    string str = @"-Export " + Dir + ParentLayerName + ".stp" + " _enter";
    RhinoApp.RunScript(str, " ", false);
  }
